## üì¶ Getting Started

### Prerequisites

- Flutter SDK (Latest Stable)
- firebase-tools (for manual deployment)

### Installation

1. **Clone the repository**:

   ```bash
   git clone https://github.com/Digvijaysinh2204/portfolio.git
   ```

2. **Navigate to project directory**:

   ```bash
   cd portfolio
   ```

3. **Install dependencies**:

   ```bash
   flutter pub get
   ```

4. **Run the project**:
   ```bash
   flutter run -d chrome
   ```

---

## üö¢ Deployment

The project is configured with GitHub Actions. Any push to the `main` branch will automatically trigger:

1. Flutter Environment Setup (with caching for speed)
2. Web Release Build
3. Automated Deployment to **Firebase Hosting**
